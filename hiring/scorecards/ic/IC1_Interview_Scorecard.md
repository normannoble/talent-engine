## Level Context: What IC1 Looks Like

IC1 is an **entry-level role** focused on learning and building foundations. Evaluate for:

- **Potential over polish** — Look for learning ability, not expertise
- **Foundational skills** — Basic programming concepts, not advanced patterns
- **Coachability** — How they respond to guidance and feedback
- **Curiosity** — Genuine interest in learning and growing

**Do NOT expect:** Deep expertise, complex system design, or extensive experience.

---

## Competency Assessment

### 1. Technical Foundation

**What to assess:** Basic programming knowledge, understanding of fundamentals, ability to learn.

**IC1 Expectations:**
- Understands basic programming concepts (variables, loops, functions, data structures)
- Can write simple code with guidance
- Demonstrates logical thinking
- Shows ability to learn and apply new concepts

**Sample Questions:**
- Walk me through how you would solve [simple coding problem].
- Explain what happens when [basic concept].
- Tell me about a technical concept you learned recently. How did you learn it?
